# Familia: Formicidae

Formicidae es la única familia dentro del orden Hymenoptera que comprende a todas las hormigas conocidas. Con más de 20,000 especies descritas, es uno de los grupos de insectos más diversos y ecológicamente dominantes del planeta.

## Datos generales

| Campo | Valor |
|-------|-------|
| Orden | Hymenoptera |
| Familia | Formicidae |
| Subfamilias principales | Formicinae, Myrmicinae, Ponerinae, Dolichoderinae, Dorylinae |
| Distribución | Global (excepto Antártida) |
| Especies descritas | ~20,000+ |

## Subfamilias incluidas en este repositorio

| Subfamilia | Características destacadas | Géneros documentados |
|------------|---------------------------|----------------------|
| [Formicinae](generos/) | Glándula metapleural ausente, ácido fórmico | Camponotus, Brachymyrmex |
| Myrmicinae | Pecíolo + postpecíolo, aguijón funcional | Monomorium, Messor, Acanthomyrmex, Pogonomyrmex, Crematogaster, Pheidole, Solenopsis |
| Dolichoderinae | Solo pecíolo, sin aguijón, secreciones volátiles | Dorymyrmex |
