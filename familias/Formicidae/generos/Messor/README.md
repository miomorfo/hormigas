# Género: Messor

Género de hormigas granívoras ("cosechadoras") de la subfamilia Myrmicinae, altamente adaptadas a climas áridos y semiáridos. Son famosas por recolectar y almacenar semillas en cámaras especiales del nido llamadas graneros, y por su marcado polimorfismo con soldados de cabeza enorme.

## Datos generales

| Campo | Valor |
|-------|-------|
| Familia | Formicidae |
| Subfamilia | Myrmicinae |
| Género | *Messor* Forel, 1890 |
| Distribución | Mediterráneo, África, Asia, sur de Europa |
| Hábitat | Zonas abiertas, pastizales, campos agrícolas, laderas secas |

## Características del género

- **Dieta:** Principalmente granívora — recolectan y almacenan semillas.
- **"Pan de hormiga":** Trituran las semillas con sus mandíbulas para crear una pasta nutritiva.
- **Polimorfismo:** Marcado. Soldados con cabeza desproporcionadamente grande para triturar semillas.
- **Nidificación:** Suelo, nidos profundos con cámaras diferenciadas (graneros secos + cría húmeda).
- **Estructura social:** Generalmente monóginas.
