# Género: Pheidole

Uno de los géneros más grandes de hormigas del mundo (~1,000 especies). Conocidas como "hormigas cabezonas" por el marcado dimorfismo entre obreras menores y soldados de cabeza enorme. Subfamilia Myrmicinae.

## Datos generales

| Campo | Valor |
|-------|-------|
| Familia | Formicidae |
| Subfamilia | Myrmicinae |
| Género | *Pheidole* Westwood, 1839 |
| Distribución | Global (cosmopolita) |
| Especies descritas | ~1,000+ |

## Características del género

- **Dimorfismo marcado** — obreras menores pequeñas + soldados (majors) con cabeza desproporcionadamente grande.
- **Crecimiento rápido** — una de las más veloces del hobby.
- **Muy agresivas** — excelentes reclutadoras, atacan presas en masa.
- **Dieta:** Omnívoras generalistas.
- **Nidificación:** Suelo, bajo piedras, madera muerta.
