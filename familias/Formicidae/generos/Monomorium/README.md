# Género: Monomorium

Género grande y diverso de la subfamilia Myrmicinae con ~297 especies descritas. Incluye algunas de las hormigas vagabundas más exitosas del mundo (*M. pharaonis*) y varias especies endémicas de Chile.

## Datos generales

| Campo | Valor |
|-------|-------|
| Familia | Formicidae |
| Subfamilia | Myrmicinae |
| Género | *Monomorium* Mayr, 1855 |
| Distribución | Global; centros de especiación en África y Australia |
| Especies descritas | ~297 + 22 subespecies |
| Tamaño medio de colonia | Decenas a miles de individuos |

## Características del género

- **Tamaño:** Pequeñas a medianas.
- **Antenas:** 10–12 segmentos con maza gradual de 3–4 segmentos.
- **Polimorfismo:** Mayormente monomorfas; algunas especies polimórficas.
- **Nidificación:** Bajo piedras, suelo.
- **Dieta:** Omnívoras generalistas; algunas granívoras.
- **Estructura social:** Generalmente poligínicas (varias reinas por colonia).
