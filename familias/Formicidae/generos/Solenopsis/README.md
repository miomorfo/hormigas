# Género: Solenopsis

Género de hormigas pequeñas de la subfamilia Myrmicinae. Incluye tanto las famosas "hormigas de fuego" tropicales como especies diminutas crípticas del subgénero *Diplorhoptrum*. En Chile solo se encuentran especies del grupo pequeño.

## Datos generales

| Campo | Valor |
|-------|-------|
| Familia | Formicidae |
| Subfamilia | Myrmicinae |
| Género | *Solenopsis* Westwood, 1840 |
| Distribución | Global (cosmopolita) |
| Especies descritas | ~200+ |
| Especies en Chile | ~6 |

## Características del género

- **Tamaño:** Variable — desde 1 mm (subgénero *Diplorhoptrum*) hasta 6 mm (hormigas de fuego).
- **Antenas:** 10 segmentos con maza de 2 segmentos — rasgo diagnóstico.
- **Aguijón:** Funcional, picadura dolorosa en las especies de fuego; leve en las pequeñas.
- **Dieta:** Omnívoras generalistas.
- **Nidificación:** Suelo, bajo piedras; especies pequeñas frecuentemente subterráneas.
